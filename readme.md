## FastTask

### desc
  科技以套壳为本
  
  使用fastapi 对celery的最简单封装, 仅暴露5个任务简单的接口以及两个文件接口(upload, download)， 具体调用可以拉取```irid/fasttask```镜像查看接口文档

  1. 创建任务 -- create
  2. 检测任务(获取结果) -- check
  3. 同步调用任务 -- run
  4. 文件上传接口 -- upload
  5. 文件下载接口 -- download
